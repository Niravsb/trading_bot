# Binance Futures Testnet Trading Bot

A simplified CLI-based Python application to place MARKET and LIMIT orders on the Binance Futures Testnet (USDT-M).

## Setup Steps
1. Register for a Binance Futures Testnet account and generate your API keys.
2. Clone this repository or extract the source code.
3. Install the required dependencies:

   pip install -r requirements.txt
